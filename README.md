# CircleTODO——用色彩标记日程！

---

## 特色功能 ##
CircleTODO是一款新颖的日程管理软件，除了具有独特的以色彩标记日程的功能外，还可以将日程备份至云端，这样可以非常方便地多终端同步日程，CircleTODO还支持分享标记，您可以将自己的标记分享给其他人，应用于布置工作将会非常的方便。
